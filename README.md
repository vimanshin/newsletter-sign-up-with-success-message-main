## Обзор

### Задание

Пользователи должны иметь возможность:

- Ввести свой email и отправить форму
- После успешной отправки формы увидеть сообщение со своим email
- Увидеть сообщения об ошибках валидации формы, если:
  - Поле осталось пустым
  - Адрес электронной почты имеет неправильный формат
- Видеть оптимальную компоновку интерфейса в зависимости от размера экрана устройства
- Видеть состояния `hover` и `focus` для всех интерактивных элементов страницы
